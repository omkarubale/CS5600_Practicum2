# CS5600_Practicum2

Make the Project:
>> make clean
>> make 

Run the server:
>> cd server
>> ./server

To run the test file:

// Navigate to Client
>> cd ../client
>> ./testing

To run individual commands on the client:
// No. of arguments as per the command.
>> ./fget COMMAND ARG1 ARG2 

eg: ./fget GET h5.txt f1/h2.txt
